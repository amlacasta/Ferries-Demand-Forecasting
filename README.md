Forecasting de Demanda por Salida (Trip-Level) para Control Operativo
🚢 LevanteFerries: Optimización de Capacidad y Estrategia de Ingresos
Este proyecto desarrolla un sistema de Forecasting de Demanda de alta resolución (nivel trayecto) para una naviera de ferries. El objetivo es transformar datos históricos y curvas de reserva en una Torre de Control operativa que permita anticipar saturaciones o infrautilizaciones de flota.

📝 Contexto y Problema
En el transporte marítimo de pasajeros, la demanda es extremadamente volátil debido a:

Estacionalidad: Contrastes críticos entre temporada alta (verano) y baja (invierno).

Calendario: Picos en fines de semana y festivos nacionales/regionales.

Factores Externos: Condiciones meteorológicas (mala mar) y retrasos operativos.

Riesgos identificados:

Saturación (Stock-out): Pérdida de ingresos al alcanzar el 100% de ocupación prematuramente.

Infrautilización: Salidas con baja ocupación que elevan el coste unitario por plaza y erosionan el margen operativo.
